# 69SPLIT

(image)

A split keyboard with 69 keys made for people that like to expose their I/O for any usecase.

* Keyboard Maintainer: [Yiannis Michael](https://github.com/ymich9963)
* Hardware Supported: rev1
* Hardware Availability: N/A

Make example for this keyboard (after setting up your build environment):

    make 69SPLIT

Flashing example for this keyboard:

    make 69SPLIT:default:flash

See the [build envronment setup](https://docs.qmk.fm/newbs_getting_started) and the [make instructions](https://docs.qmk.fm/getting_started_make_guide) for more information. Brand new to QMK? Start with our [Complete Newbs Guide](https://docs.qmk.fm/newbs).

## Bootloader

Enter the bootloader with the RESET button.
